05 · For the dev team

The design system.

Colour, type, spacing, components, motion. Hand this to a developer and they can ship the site brand-accurate.

01

Colour tokens

Mint#05CEAAAccent. Emphasis in headlines, CTAs, icon fills, mint rules.
Brick#BA2D0BTension signal only — problems, warnings, "before". Never body.
Onyx#131313Primary type, dark sections, nav pill base, hero background.
Paper#F4F4F9Default page background. Always paper-first, dark used sparingly.
Line#EAEAEEHairline borders, dividers, card outlines.
Muted#8A8A92Captions, labels, secondary meta text.

Do

  • Lead with Paper, accent with Mint, tension with Brick, anchor with Onyx.
  • Use Mint on ~10–15% of surface area — it loses punch above that.
  • Reserve Brick for "the problem" framing; not a decorative colour.

Don't

  • Use Mint and Brick in the same element or adjacent blocks.
  • Introduce new hues — gradients, pastels, alternate greens.
  • Set body text in Mint or on coloured fills; use Onyx on Paper.

Do

  • Use only Poppins 300 (Light) and 600 (SemiBold). Two weights, never more.
  • Write in sentence case. Allcaps is reserved for Labels and Captions.
  • Highlight one phrase per heading in Mint via .emph. Never two.

Don't

  • Mix in Regular/Medium/Bold — it breaks the editorial rhythm.
  • Use Title Case or ALL CAPS on H1/H2 headings.
  • Stack multiple Mint emphases in the same sentence.
03

Grid, spacing, rhythm

Container

Max-width 1280 · edge 40

Every section centers on .container{max-width:1280px;padding:0 40px;}. Mobile drops to 24px edge.

Grid

12-column, 24px gutter

Home stats use asymmetric ratios 1.6fr 1fr 1fr. Paths and press use repeat(3,1fr). Team goes masonry.

Section padding

96 / 120 / 140px vertical

Standard 96px. Hero-adjacent stats use 120px. Closing-moment sections 140px. Mobile halves.

Card padding

28–36px

Quote / path / press cards sit on 32px padding and 6px border radius. Audit tiles: 28–32px.

Radius

4px everywhere — 999px for pills

Flat 4px corners on cards, figures, tiles. Circles only for nav pill, CTA buttons, icon badges, avatars.

Hairlines

1px Line (#EAEAEE)

Borders, dividers, and top-rules use one shade. Never heavier than 1px for structure.

Floating pill nav

Two pills, centered, fixed at top with 24px inset. Logo dot is Mint inside a white pill. CTA pill is Onyx with Mint dot.

Icon badge

↗ ◆ ★

Mint-filled circle, Onyx icon inside (never inverted). 44 / 56 / 64px sizes. Hover: scale(1.06) rotate(-3deg).

05

Imagery

Portraits

B&W, grayscale(1)

Every named portrait is desaturated — filter:grayscale(1) contrast(1.05). Mint is the only colour on the page.

Ratio

4:5 or 1:1

Editorial 4:5 for hero figures and story blocks. 1:1 for team tiles. 2:1 horizontal strips for quote takeovers.

Crop

Top-weighted, object-position: center 22%

Keep eyes in the upper third. Faces never centered — editorial, not ID-card.

06

Motion

Scroll reveal

Fade + 24px rise

.reveal becomes .in via IntersectionObserver (threshold .15). Duration 700ms, ease cubic-bezier(.2,.6,.2,1). Delays via .delay-1/2/3 (80/160/240ms).

Word reveal

Hero H1 only

.word-reveal wraps each word in a .w span. Staggered 60ms per word. Used only on the home hero H1.

Marquee

40s linear, pauses on hover

Content duplicated to 200% width, animated to translateX(-50%). Reserved for the Momentum ticker.

Hover

4px lift + shadow

Cards: translateY(-4px) + 0 18px 48px rgba(19,19,19,.1), 220ms. Icons: scale(1.06) rotate(-3deg).

Reduced motion

prefers-reduced-motion

All reveals, word-reveals, and marquee are disabled. Content still appears, instantly, with no transforms.

Easing

Two curves, everywhere

Slow reveal = cubic-bezier(.2,.6,.2,1). Hover = 220ms same curve. Nothing bounces, nothing overshoots.

07

Voice & copy rules

Do

  • Write like a senior operator. Short sentences, precise nouns, zero jargon.
  • Name people. Use real quotes. Credit the source.
  • End sections with one clear next step — never two.

Don't

  • Claim "diversity" — claim proof. (Names, tenures, sectors.)
  • Use stacked adjectives ("innovative, dynamic, forward-thinking").
  • Use exclamation marks, emoji, or marketing-ese ("unlock", "elevate", "synergy").
